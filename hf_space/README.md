# Pokemon Battle Predictor

Application Streamlit de prediction du taux de victoire Pokemon.

## Features

- Saisie des statistiques principales
- Prediction d'un taux de victoire
- Recommandation rapide (Tres fort, Fort, Moyen, Faible)

## Notes

- Si le fichier modele n'est pas present, l'application utilise un score de secours.
- Pour de meilleures predictions, ajouter `modele/modele_pokemon.mod` au Space.
